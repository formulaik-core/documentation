## Formulaik Generator 
The Formulaik generator bootstraps ready to use Formulaik projects.
It uses Formulaik itself with the Chunks extension.

[Github](https://github.com/formulaik-core/generator)

<BashTabs npm={`npm install -g formulaik`} />
Or
<BashTabs npm={`npx formulaik`} />
