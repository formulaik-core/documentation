# Question

- Project wide question in the form of a _formulaik file in <b>src/questions/</b>.
- To inherit a question the only required argument is ```name```. The rest of the arguments will be inherited.
- Order of precedence: **Formulaik.prompt.ask** > **command** > **src/questions**
- A **Formulaik.prompt.ask** question inherits a matching **command** question if there's any
- If there's no matching **command** question, **Formulaik.prompt.ask** question inherits a matching **src/questions** question if there's any
- A **command** question inherits a **src/questions** question if there's any
- A **Formulaik.prompt.ask** question inherits a **command** question
- A question defined by **Command** takes precedence over a question defined in **Command**

## Usage

You can pass the name of the transformer in a command <i>questions</i>.

In any **Command** (```src/commands``` folder)
```js
export default ({
    _formulaikType: "command",
    name: 'schema',
    questions: [
    {
        name: 'appPath',
        message: "Path to app",
        validators: [{ id: 'nonempty' }]
    },
]
})
```

In a command handler **Formulaik.prompt.ask**
```js
await Formulaik.prompt.ask([
    {        
        name: 'appPath',
        message: "Path to app",
        validators: [{ id: 'nonempty' }]        
    },
])
```
