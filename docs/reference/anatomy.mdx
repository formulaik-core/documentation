# Anatomy of a Formulaik app

## Project structure
```bash
├── src
├──── app.js
├──── commands
├────── command A
├──────── sub command X
├──────── sub command Y
├────── command B
├──────── sub command Z
├──────── sub command W
├──── extensions
├──── questions
├──── transformers
├──── validators
├── formulaik.config.js
├── .env
└── package.json
```

- The formulaik config is formulaik.config.js
- The formulaik app entry point is index.js
- The local extensions are in /src/extensions
