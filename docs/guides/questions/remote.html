Remote questions

You can ask a series of questions by providing a JSON array fetched remotely or loaded from a local file. This can be useful when the questions to ask can not be predetermined.

Exampleâ€‹

We want to ask questions depending on the type of engine the user chooses. The questions are available on our server with a GET request.

import axios from "axios"

const askRemoteQuestions = async ({ id }) => {
    const url = `https://myserver.com/questions`  
    const result = await axios({
        method: "GET",
        url,
        headers: {
            "content-type": "application/json"
        },
        params: {
            id,        
        }
    })

    await Formulaik.prompt.ask(result.data)  
}
