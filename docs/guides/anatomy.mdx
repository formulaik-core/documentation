# Anatomy of a Formulaik app

## Project structure
```bash
├── app.js
├── src
├──── app.js
├──── commands
├────── command A
├──────── sub command X
├──────── sub command Y
├────── command B
├──────── sub command Z
├──────── sub command W
├── formulaik.config.js
├── .env
└── package.json
```

- The formulaik config is formulaik.config.js
- The formulaik app entry point is index.js
- The app feature is in /app
- The local features are in /features
