Formulaik
for React

An open source framework for building modern forms. Formulaik extends Formik with a JSON based declarative approach 🚀

Quick StartGuides

Declarative

Formulaik makes it easy to add a command by creating a folder and adding its subcommands.

Input libraries driven

Options can be defined at a global level and inherited at a command level to promote reuse.

Extensible

Every aspect of a command parameters can be declared easily via input and output transformers, an input validators.

Quick start

1. Add Formulaik:

  • npm
  • yarn
npm install @formulaik/react
yarn add @formulaik/react

Ask smart questions 🙋‍♀️

Every command can be given questions with powerful side effects, transformers and validators. Furthermore Formulaik automatically extracts options from questions to build the CLI helper.

Validators

When features need to communicate the Formulaik.Services namespace is ready for inter-feature communication. Any feature can register an api associated with a unique ID that can be called with no knowledge of the feature that handles the operation.

Transformers

Formulaik Automatic Seed is an easy way to seed Formulaik models. It was built from the ground up to respect the feature lifecycle and separation of concerns. Every feature can define config entries that drive its behaviour at runtime, including the app feature.

Side effects

Formulaik config adds the ability to change the behaviour and appearance of your Formulaik app without redeploying your server or requiring users to update their app. Every feature can manage its own configuration.

We don't want to spend too much time on the CLI either 🤖

Formulaik is built from the ground up to be easy to use, convention driven and extensible so that it's easy to kickstart a project and maintain it. It's api makes it easy to fetch steps remotely and execute them with powerful side effects, transformers and validators.
